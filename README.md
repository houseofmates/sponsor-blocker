# sponsor-blocker

a browser extension that automatically skips sponsored segments in youtube videos using crowd-sourced data.

## what is this?

a browser extension that automatically skips sponsored segments in youtube videos using crowd-sourced data.

this is a fork of [ajayyy/SponsorBlock](https://github.com/ajayyy/SponsorBlock) with a redesigned ui.

## original vs. this version

this is a fork of the [ajayyy/SponsorBlock](https://github.com/ajayyy/SponsorBlock) project.

### changes from original

- redesigned the ui with a dark pkm aesthetic. all text is now lowercase. extension renamed from sponsorblock to sponsor-blocker. auto-update checking added.

## installation

download the `.xpi` file from releases and open it in firefox, or load it temporarily via `about:debugging` > this firefox > load temporary add-on.
